# 📊 Paisa Meter - All-in-One Financial Toolkit

**Paisa Meter** ek modern, fast aur user-friendly web application hai jo logon ko unke rozana ke financial calculations mein madad karta hai. Is platform ko simple aur accurate banane ke liye design kiya gaya hai.

## 🚀 Features
* **EMI Calculator:** Ghar, gaadi ya personal loan ki kist turant check karein.
* **SIP Calculator:** Apne mutual fund investments par milne wale returns ka anuman lagayein.
* **Age Calculator:** Apni sateek umar (Saal, Mahine, Din) pata karein.
* **Universal Currency Converter:** Duniya bhar ki currencies ko live rates par badlein.
* **Dark Mode Support:** Raat mein istemal ke liye aankhon ko sukoon dene wali theme.
* **Fully Responsive:** Mobile, Tablet aur Desktop par perfectly chalti hai.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Charts:** Chart.js (Interactive visual data ke liye)
* **Hosting:** Vercel / GitHub Pages
* **SEO:** Optimized Meta Tags for better Google Ranking
